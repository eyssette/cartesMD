# CartesMD / FlashMD

CartesMD est un outil libre et gratuit qui permet de générer des cartes à jouer à partir d'un simple texte en Markdown.

FlashMD est une version simplifiée de CartesMD qui permet de créer des flashcards sur le même principe, mais avec une syntaxe plus simple.

Les deux outils sont déployés à partir du même code source, et sont disponibles en ligne à l'adresse suivante :

- [https://cartesmd.forge.apps.education.fr/](https://cartesmd.forge.apps.education.fr/).
- [https://flashmd.forge.apps.education.fr/](https://flashmd.forge.apps.education.fr/).
